# WaterBalance 🌊

Минималистичное iOS-приложение для контроля водного баланса. 
### 📱 Что внутри:
* **Кастомный UI:** Индикатор прогресса в виде аккуратного заполняющегося кольца с градиентом и плавной анимацией.
* **Сохранение прогресса:** Данные не сбрасываются при закрытии приложения — использовала `UserDefaults`.
* **Архитектура:** Логика подсчета и хранения вынесена в отдельный менеджер (`WaterDataManager`), чтобы не перегружать интерфейс.

### 🛠 Стек:
* Swift, SwiftUI, Combine (ObservableObject).

### 🚀 Как запустить:
1. Клонировать репозиторий.
2. Открыть проект в Xcode.
3. Запустить на симуляторе или прямо в Xcode Preview.
